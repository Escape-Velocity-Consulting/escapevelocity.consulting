Generators

Script Command Purpose
build:tokens tsx scripts/build-tokens.ts Regenerate tokens.css from tokens.ts
sync:website bash scripts/sync-website.sh Copy tokens.css → website/styles/
pdf tsx generators/pdf.ts Markdown → branded A4 PDF
image tsx generators/image.ts HTML/SVG → PNG (banners, social)
export tsx scripts/export-assets.ts Regenerate all raster assets + previews
build:skill bash scripts/build-skill.sh Package brand-engine.skill

Token Update Workflow

1

Edit brand/tokens.ts — Change colors, fonts, or spacing values

2

Run npm run build:tokens — Regenerates brand/tokens.css from source

3

Run npm run sync:website — Copies tokens.css into website/styles/tokens.css

4

Commit both repos — Brand repo and website repo are separate git repos

Skill Packaging

The brand system can be packaged as a Claude Code skill for agent use.

npm run build:skill
# Output: dist/brand-engine.skill
